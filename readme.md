## What We Do

We are a technology company based in Surat, India. Our work spans three core directions —

**🎮 HTML5 Games**
Building a web-native gaming platform where games run instantly in any browser, on any device. No downloads. No friction.

**📱 Android Applications**
Crafting native Android experiences with modern architecture — fast, reliable, and built to scale.

**🤖 Artificial Intelligence**
Running active research and development in AI — focusing on user behavior, automation, and intelligent decision-making systems.
